This project has three external library dependencies:

1. `ActionBarSherlock` to add actionbar support to API < 11 (pre-honeycomb).
    
        http://actionbarsherlock.com/
        Version 4.2.0

2. `Google Play services` for maps display

       https://developer.android.com/google/play-services/
       Version 2.0.10

3. `Gson` to convert JSON to Java objects

       http://code.google.com/p/google-gson/
       Version 2.2.2
